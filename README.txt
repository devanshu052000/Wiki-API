Wiki-API

Developed a Rest API for wikipedia with a small database.

Motivation

Developed it while completing the course "The Complete 2022 Web Development Bootcamp". Aim was to learn creating RESTful APIs and route chaining.

Tech/Framework used

NodeJS, ExpressJS, MongoDB, and Mongoose.

Installation

Install all the dependencies by typing the command "npm install" inside command line.
